개선 포인트
장바구니 통합 뷰 P0
공동 포토북 → PC 상품 → 모바일 상품 순으로 그룹 통합 노출. 기기 상관없이 전체 장바구니 한눈에 확인.
기기 제한 상품 처리 P0
편집 불가 상품도 구매는 가능. 편집 버튼만 비활성화하고 체크박스·주문은 정상 동작. 포토북은 예외로 모든 기기에서 편집 허용.
쿠폰 만료 인라인 표시 P0
24h 미만 시:분:초 · 3일 이내 D-N. 상품 카드 하단 앰버 뱃지로 긴급감 전달.
최대 혜택 할인 자동 적용 P0 NEW
가능한 최대 할인을 시스템이 자동 적용. 프로젝트별 그린 뱃지 + 장바구니 전체 상단 배너·요약에 "최대 혜택 적용중" 노출. ※ 최적 할인 자동선택 로직은 개발 검토 필요
상품 할인가 강조 P1 NEW
할인 상품에 정가 취소선 + 강조가. 시안 A 할인율(10%) · 시안 B 절약액(2,312원 할인) — 카드에서 두 안 비교 후 택1.
무료배송 안내 배너 P1
잔여 금액 실시간 표시 + 포토북 포함 시 무료배송 조건 안내.
절약액 동기 부여 P1
주문 버튼에 절약 금액(앰버) 표기. 최종 결제 예정금 + 원가를 동시에 노출.
배송지 가독성 개선 P0
저장된 배송지가 있으면 1–2줄 요약 카드. 이름·연락처·주소 한눈에 확인, 변경도 1탭.
결제수단 순서 변경 P0
간편결제(네이버·토스·카카오) 최상단 배치. 마지막 사용 결제수단 자동 선택 유지.
주문자 정보 최소화 + 알림톡 P0 NEW
주문자 입력란 제거. 본인인증 완료 → 인증 번호로 알림톡, 미입력. 미인증·비회원 → 배송지 연락처로 발송. ※ 선물 주문(수령인=배송지) 시 알림 수신처 협의 필요
결제수단 혜택 안내 P1 NEW
혜택 제공 결제수단(예: 네이버페이) 선택 시 적립·할인 혜택을 인라인 안내. ※ 적립률·조건 수치는 제휴 확인 후 확정(현재 예시)
약관 동의 간소화 P1
항목 통합 + 결제 버튼 하단 소형 안내 텍스트로 마찰 최소화. [법무 검토 필요]
