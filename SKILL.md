# Document Summarizer

## Purpose

Help users create clear, concise summaries of long documents and texts.

## Instructions

When summarizing documents:

### 1. Summary Types

Offer appropriate summary:

**Executive Summary:**
- High-level overview
- Key conclusions first
- Decision-focused
- 1-2 pages max

**Abstract:**
- Purpose and scope
- Methodology
- Key findings
- Conclusions

**Bullet Summary:**
- Key points only
- Quick reference
- Scannable format

**Detailed Summary:**
- Section by section
- More comprehensive
- Preserves structure

### 2. Summary Process

Create effective summaries:

1. Identify document purpose
2. Find main arguments/points
3. Note key data/evidence
4. Capture conclusions
5. Identify action items

### 3. What to Include

Essential elements:

- Main thesis/purpose
- Key findings
- Important data points
- Conclusions
- Recommendations
- Action items

### 4. What to Exclude

Remove unnecessary:

- Excessive details
- Redundant information
- Minor examples
- Technical jargon (unless needed)
- Filler content

### 5. Length Guidelines

Match summary to purpose:

- Quick brief: 1-3 sentences
- Short summary: 1 paragraph
- Standard: 1 page (10%)
- Detailed: 2-3 pages (15-20%)

## Output Format

Provide summary with:

1. One-sentence overview
2. Key points section
3. Important details
4. Conclusions/recommendations
